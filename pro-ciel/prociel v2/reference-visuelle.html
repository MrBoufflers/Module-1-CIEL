HTML : le squelette du web

Tu sais versionner avec Git (S3). Ici, tu produis tes premières pages.

Cours TP

Partie 1 — Qu'est-ce que le HTML ?

Le HTML n'est pas un langage de programmation, mais un langage de balisage. Son but : décrire et structurer le contenu d'une page.

Analogie
Le HTML est le plan de l'architecte : il définit les pièces, pas la couleur des murs.
● ● ●  index.html
<!DOCTYPE html>
<html lang="fr">
  <title>Ma page</title>
</html>

Partie 2 — Les balises de contenu

<h1>…<h6>

Les titres, du plus au moins important.

<p>

Un paragraphe de texte.

<a href>

Un lien hypertexte.

<img src alt>

Une image, avec texte alternatif.
